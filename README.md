### `Memory Game React`
A small memory game created by using react & typescript

### Installation
git clone git@github.com:Fanz0614/memory_game.git

### Run the demo
cd memory_game
npm start

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
